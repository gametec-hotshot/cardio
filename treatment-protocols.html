Evidence-Based Treatment Protocols

Comprehensive therapeutic algorithms, medication guidelines, and clinical protocols based on latest cardiovascular research and professional society recommendations.

Treatment Categories

Heart Failure Treatment Protocol

ACC/AHA 2022 Guidelines

HFrEF (EF ≤ 40%) Treatment Algorithm

1

First-Line Therapy

Class I Recommendation
ACE Inhibitors
  • • Enalapril: 2.5-20mg BID
  • • Ramipril: 1.25-10mg daily
  • • Lisinopril: 5-40mg daily
  • • Target: Max tolerated evidence-based dose
Beta-Blockers
  • • Carvedilol: 3.125-25mg BID
  • • Metoprolol succinate: 12.5-200mg daily
  • • Bisoprolol: 1.25-10mg daily
  • • Titrate to max tolerated dose
Clinical Pearls
  • • Start ACE inhibitor first, then add beta-blocker once stable
  • • Titrate both medications to maximum tolerated evidence-based doses
  • • Monitor renal function and electrolytes with each dose change
  • • Continue diuretics as needed for symptom control
2

Second-Line Therapy

Class I Recommendation
MRA
  • • Spironolactone: 25mg daily
  • • Eplerenone: 25mg daily
  • • Monitor K+ and creatinine
SGLT2 Inhibitors
  • • Dapagliflozin: 10mg daily
  • • Empagliflozin: 10mg daily
  • • Reduces CV death and HF hospitalization
ARNI Alternative
  • • Sacubitril/valsartan: 24/26mg BID
  • • Superior to ACE inhibitors
  • • 36-hour washout required
3

Third-Line/Specialist Referral

Class IIa Recommendation
Ivabradine
  • • Starting: 5mg BID
  • • Target: 7.5mg BID
  • • HR ≥75 bpm, sinus rhythm
  • • Reduces CV death and HF hospitalization
Device Therapy
  • • ICD if EF ≤35% and NYHA II-III
  • • CRT if QRS ≥150ms and LBBB
  • • Consider cardiac transplantation

HFpEF (EF ≥ 50%) Treatment

Pharmacologic Management

  • Diuretics: For volume management and symptom relief
  • SGLT2 Inhibitors: Dapagliflozin or empagliflozin reduce HF hospitalizations
  • ARB/ARNI: Consider for hypertension management

Comorbidity Management

  • Hypertension: Target <130/80 mmHg
  • Diabetes: Optimize glycemic control
  • Atrial Fibrillation: Rate and rhythm control

Coronary Artery Disease Treatment

AHA/ACC 2021 Guidelines

Medical Management

Antiplatelet Therapy

  • • Aspirin 81mg daily (lifelong)
  • • Clopidogrel 75mg daily if aspirin contraindicated
  • • Dual antiplatelet therapy post-PCI/stent

Statin Therapy

  • • High-intensity statin for all patients
  • • Atorvastatin 40-80mg daily
  • • Rosuvastatin 20-40mg daily
  • • Target LDL-C <70 mg/dL

Beta-Blockers

  • • Metoprolol succinate 25-200mg daily
  • • Carvedilol 3.125-25mg BID
  • • Titrate to resting HR 55-60 bpm

Revascularization Strategy

PCI (Percutaneous Coronary Intervention)

  • • Primary PCI for STEMI within 90 minutes
  • • Elective PCI for high-risk anatomy
  • • Drug-eluting stents preferred
  • • Dual antiplatelet therapy 6-12 months

CABG (Coronary Artery Bypass Graft)

  • • Left main disease (>50% stenosis)
  • • Three-vessel disease with diabetes
  • • Two-vessel disease with proximal LAD
  • • SYNTAX score >22

Hypertension Treatment Protocol

AHA/ACC 2017 Guidelines
1

Lifestyle Modifications

Class I Recommendation
Dietary Changes
  • • DASH diet: 8-14 mmHg reduction
  • • Sodium restriction: <2,300mg daily
  • • Limit alcohol: ≤2 drinks (men), ≤1 (women)
  • • Increase potassium intake
Physical Activity
  • • Aerobic exercise: 30 min, 5-7 days/week
  • • Resistance training: 2-3 days/week
  • • Weight loss if overweight
  • • Smoking cessation
2

Pharmacologic Therapy

Class I Recommendation
First-Line Agents
  • • Thiazide diuretics
  • • ACE inhibitors
  • • ARBs
  • • Calcium channel blockers
Second-Line Agents
  • • Beta-blockers
  • • Aldosterone antagonists
  • • Alpha-blockers
  • • Direct vasodilators
Target Goals
  • • <130/80 mmHg (most patients)
  • • <130/80 mmHg (diabetes)
  • • <120/80 mmHg (high cardiovascular risk)
  • • Individualized targets

Atrial Fibrillation Treatment Protocol

AHA/ACC/HRS 2023 Guidelines
1

Rate vs Rhythm Control

Class I Recommendation
Rate Control
  • • Beta-blockers (first-line)
  • • Non-DHP calcium channel blockers
  • • Digoxin (if HFrEF)
  • • Target HR <110 bpm (lenient)
Rhythm Control
  • • Consider if symptomatic
  • • Antiarrhythmic drugs
  • • Catheter ablation
  • • Electrical cardioversion
2

Anticoagulation

Class I Recommendation
CHA₂DS₂-VASc Score
  • • Score ≥2: Anticoagulation
  • • Score 1: Consider anticoagulation
  • • Score 0: No anticoagulation
  • • Assess bleeding risk
Anticoagulant Options
  • • DOACs (first-line)
  • • Warfarin (if mechanical valve)
  • • Left atrial appendage closure
  • • Aspirin (limited role)

Dyslipidemia Treatment Protocol

AHA/ACC 2018 Guidelines
1

Statin Therapy

Class I Recommendation
High-Intensity
  • • Atorvastatin 40-80mg
  • • Rosuvastatin 20-40mg
  • • LDL-C reduction ≥50%
  • • ASCVD risk ≥20%
Moderate-Intensity
  • • Atorvastatin 10-20mg
  • • Rosuvastatin 5-10mg
  • • Simvastatin 20-40mg
  • • LDL-C reduction 30-49%
Lifestyle
  • • Heart-healthy diet
  • • Regular exercise
  • • Weight management
  • • Smoking cessation
2

Non-Statin Therapies

Class IIa Recommendation
PCSK9 Inhibitors
  • • Evolocumab 140mg every 2 weeks
  • • Alirocumab 75-150mg every 2 weeks
  • • LDL-C reduction 50-60%
  • • Familial hypercholesterolemia
Other Agents
  • • Ezetimibe
  • • Bempedoic acid
  • • Bile acid sequestrants
  • • Fibrates (if high triglycerides)

Acute Coronary Syndrome Protocol

AHA/ACC 2020 Guidelines
🚨

Immediate Management

Emergency
STEMI (ST-Elevation MI)
  • • Primary PCI within 90 minutes
  • • Dual antiplatelet therapy
  • • Anticoagulation
  • • Statin therapy
  • • Beta-blocker
NSTEMI/Unstable Angina
  • • Risk stratification (TIMI, GRACE)
  • • Early invasive if high risk
  • • Conservative if low risk
  • • Dual antiplatelet therapy
  • • Anticoagulation
1

Antiplatelet Therapy

Class I Recommendation
Dual Antiplatelet Therapy
  • • Aspirin: 162-325mg loading, then 81mg daily
  • • P2Y12 inhibitor: clopidogrel, ticagrelor, or prasugrel
  • • Continue for 12 months minimum
  • • Consider longer duration if high bleeding risk

Venous Thromboembolism (VTE) Treatment

CHEST 2021 / ACCP Guidelines

Clinical Diagnosis & Risk Stratification Algorithm

1

Step 1 — Assess Clinical Probability

Class I
Wells Score for DVT
  • • Active cancer: +1
  • • Bedridden ≥3 days or major surgery ≤12 wk: +1
  • • Calf swelling ≥3 cm: +1
  • • Collateral superficial veins: +1
  • • Entire leg swelling: +1
  • • Localized tenderness: +1
  • • Pitting edema (affected leg): +1
  • • Paralysis/paresis/casting: +1
  • • Alternative dx ≥ as likely: −2
Score ≥2: DVT likely → ultrasound
Score <2: DVT unlikely → D-dimer first
Wells Score for PE
  • • Clinical signs/symptoms of DVT: +3
  • • PE is #1 dx (or equally likely): +3
  • • HR >100 bpm: +1.5
  • • Immobilization/surgery ≤4 wk: +1.5
  • • Prior DVT or PE: +1.5
  • • Hemoptysis: +1
  • • Malignancy: +1
>6: High risk (CTPA immediately)
2–6: Intermediate (D-dimer → CTPA)
<2: Low risk (PERC rule → D-dimer)
2

Step 2 — Confirm Diagnosis

Class I
Diagnostic Tests (DVT)
Compression Ultrasound (CUS)
First-line for all suspected proximal DVT. Sensitivity >95%.
D-dimer
Only useful in low-probability (Wells <2). High negative predictive value (>99%).
Venography
Rarely used; gold standard historically. Reserved for inconclusive cases.
Diagnostic Tests (PE)
CT Pulmonary Angiography (CTPA)
First-line for most patients with suspected PE. Sensitivity/specificity >90%.
V/Q Scan
Preferred when CTPA contraindicated (renal failure, contrast allergy, pregnancy).
⚠️ PERC Rule (0/8 required)
    Age <50, HR <100, O₂ sat ≥95%, no hemoptysis, no estrogen, no prior VTE, no surgery/trauma, no unilateral leg swelling.

Anticoagulation Management

CHEST 2016 / 2021 Updates
1

DVT Treatment

Class I
DOAC (Preferred)
  • • Rivaroxaban: 15mg BID×21d, then 20mg daily
  • • Apixaban: 10mg BID×7d, then 5mg BID
  • • No parenteral lead-in required
  • • Preferred over VKA (CHEST 2016)
LMWH + VKA
  • • Enoxaparin: 1mg/kg SC BID (or 1.5mg/kg daily)
  • • Overlap with warfarin ≥5 days
  • • Target INR 2.0-3.0
  • • Preferred if DOAC contraindicated
Duration of Therapy
  • • Provoked (transient factor): 3 months
  • • Unprovoked: ≥3 months, consider extended
  • • Recurrent VTE: Extended/indefinite
  • • Cancer-associated: ≥6 months
2

PE Treatment

Class I
Anticoagulation (Hemodynamically Stable)
  • • Same DOAC regimens as DVT
  • • Apixaban: 10mg BID×7d, then 5mg BID
  • • Rivaroxaban: 15mg BID×21d, then 20mg daily
  • • Duration same as DVT (see above)
Thrombolysis (Hemodynamically Unstable)
  • • Alteplase (rtPA): 100mg IV over 2 hours
  • • Absolute contraindications: active bleeding, recent stroke, recent surgery
  • • Half-dose (50mg) considered for select patients
  • • Rescue catheter-directed thrombolysis if systemic lytics unsuccessful
⚠️ High-Risk (Massive) PE — Emergency

SBP <90 mmHg for ≥15 min, or requiring vasopressors, or cardiac arrest.

  • • Systemic thrombolysis: alteplase 100mg IV over 2h, then heparin infusion
  • • If thrombolysis contraindicated or failed → surgical embolectomy or catheter-directed therapy
  • • ICU admission, vasopressor support as needed

Special Populations

CHEST / ESC

Cancer-Associated VTE

  • • Preferred: Apixaban or rivaroxaban (or LMWH if GI/GU risk)
  • • DOAC preferred over LMWH if no high bleeding-risk GI or GU primary tumor
  • • Minimum duration: 6 months; extend if active cancer
  • • IVC filter only if contraindication to anticoagulation

  • Reference: Key et al., CHEST 2018, 154(5):1034–1052 (ASCO 2019 update)

Pregnancy-Associated VTE

  • • Preferred: LMWH (not DOACs) throughout pregnancy
  • • Switch to UFH close to expected delivery
  • • Continue postpartum anticoagulation for ≥6 weeks (minimum 3 months total)
  • • Warfarin contraindicated in 1st trimester (teratogenic)

  • Reference: Bates et al., CHEST 2018, 154(5):1126–1183 (ACCP Guidelines)

Renal Impairment

  • • CrCl <30 mL/min: Avoid DOACs (limited data); use LMWH or UFH
  • • CrCl 15-29 mL/min: LMWH with dose adjustment (anti-Xa monitoring)
  • • On hemodialysis: consider UFH; apixaban may be used (limited data)
  • • Consider dose-reduced apixaban or rivaroxaban if CrCl 15-29 (off-label)

  • Reference: Stevens et al., CHEST 2018, 154(5):1159–1174

IVC Filter Indications

  • • Contraindication to anticoagulation (acute bleeding)
  • • Recurrent VTE despite adequate anticoagulation
  • • Retrieve filter as soon as anticoagulation is safe
  • • NOT recommended as primary treatment over anticoagulation

  • Reference: Kearon et al., CHEST 2016, 149(1):31–50 (CHEST 2021 Update)
